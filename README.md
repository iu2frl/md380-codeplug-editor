# MD380 Codeplug Editor

A static web app for editing MD380/MD390/RT3/RT8 codeplug files in the browser, with a separate local Python helper for radio read/write operations.

## Status

This project is currently in beta.

## Warning

Use this software at your own risk.

It may brick or damage your transceiver if used incorrectly, and the maintainer takes no responsibility for any device damage, data loss, or other consequences.
